Undefined Language Specification

This document defines the syntax and semantics for conditions and actions.

1. Expressions

1.1 Numeric Expressions

  • 1.1.1 NumericValue
    Represents a literal numeric value.
    Syntax: number:(value)
    Param: value — A floating-point or integer literal.
  • 1.1.2 NumericVariable
    Represents a reference to a numeric variable.
    Syntax: number_var:(name, scope)
    Params: name (Identifier), scope (global or local).
  • 1.1.3 SimpleArithmeticOperation
    Binary arithmetic operation between two numeric expressions.
    Syntax: arithmetic:(LeftExpr, operator, RightExpr)
    Valid Operators: +, -, *, /.

1.2 Boolean Expressions

  • 1.2.1 BooleanValue
    Represents a literal boolean value.
    Syntax: boolean:(value)
    Valid Values: true, false
  • 1.2.2 BooleanVariable
    Represents a reference to a boolean variable.
    Syntax: boolean_var:(name, scope)
    Params: name (Identifier), scope (global or local).
  • 1.2.3 AndOperation
    Logical AND between two boolean expressions.
    Syntax: and:(BoolExpr, BoolExpr)
  • 1.2.4 OrOperation
    Logical OR between two boolean expressions.
    Syntax: or:(BoolExpr, BoolExpr)
  • 1.2.5 NotOperation
    Logical negation of a boolean expression.
    Syntax: not:(BoolExpr)

2. Conditions

Evaluated to determine whether a trigger fires.

  • 2.1 NumericComparison
    Compares two numeric expressions using a comparator.
    Syntax: NumericExpr ; comparator ; NumericExpr
    Valid Comparators: < = > <= >= !=
  • 2.2 BooleanComparison
    Checks whether two boolean expressions evaluate to the same value.
    Syntax: BooleanExpr ; BooleanExpr
    Evaluates to true if (Left == Right).

3. Actions

Modifications or side-effects executed when a trigger activates.

  • 3.1 BooleanVariableAssignment
    Assigns a boolean expression’s value to a boolean variable.
    Syntax: BooleanVariable ; BooleanExpr
  • 3.2 NumericVariableAssignment
    Assigns a numeric expression’s value to a numeric variable.
    Syntax: NumericVariable ; NumericExpr
  • 3.3 Wait
    Pauses execution for the specified number of seconds.
    Syntax: NumericExpr
    Note: Evaluates expression as duration in seconds.
  • 3.4 ChangePosition
    Moves a game object to a new position (X and Y coordinates).
    Syntax: game_object_name ; NumericExpression ; NumericExpression
    Semantics: The first expression is X-coordinate; the second is Y-coordinate.
  • 3.5 ChangeVisibility
    Sets the visibility state of a game object.
    Syntax: game_object_name ; BooleanExpression
    Semantics: Boolean expression determines visibility (true/false).
